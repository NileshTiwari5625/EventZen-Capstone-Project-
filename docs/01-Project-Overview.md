# 01 — Project Overview

## Project Title
EventZen: Event Management System

## Problem Statement
Event planning teams need a unified platform to manage event lifecycle activities, including scheduling, venue handling, attendee communication, vendor coordination, and budget tracking.

## Proposed Solution
EventZen provides:
- Admin portal for operational management
- Customer portal for venue discovery and booking management
- Role-based access controls
- Modular architecture that supports migration from mock data to microservice-backed APIs

## Scope
- Web application with authentication and role-based authorization
- Multiple business modules beyond authentication
- Containerized deployment using Docker
